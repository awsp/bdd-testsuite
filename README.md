## My BDD Testing Suite
-------------------------------------------------
BDD Testing becomes the standard! This package is a for quick setup wrapper for BDD testing.  


### Setup
##### This will update the git submodule
```sh
sh setup.sh
```


### Running Server
```sh
sh server.sh
```
